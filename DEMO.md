# AI Form Builder - Demo Script

This demo script will guide you through showcasing the key features of the AI Form Builder, focusing on response collection and analytics.

## Demo Setup

1. **Prerequisites**
   - Ensure both backend and frontend servers are running
   - Clear any existing forms and responses for a clean demo
   - Have the application open in your browser at `http://localhost:5173`

2. **Demo Data**
   - We'll create a "Conference Feedback" form
   - Generate sample responses to demonstrate analytics

## Demo Script

### 1. Introduction (1-2 minutes)

> "Today, I'll show you our AI-powered Form Builder that makes it easy to create, distribute, and analyze surveys and forms. Let's create a feedback form for a recent conference and see how we can gather and analyze responses."

### 2. Creating a New Form (3-4 minutes)

1. **Create Form**
   - Click "Create New Form"
   - Title: "2023 Tech Conference Feedback"
   - Description: "Help us improve next year's conference!"

2. **Generate with AI**
   - Click "AI Generate Form"
   - Enter: "Conference feedback form with 5 questions about sessions, speakers, and overall experience"
   - Click "Generate Form"
   - Review the generated fields (should include session ratings, speaker feedback, etc.)
   - Click "Save Form"

### 3. Submitting Test Responses (4-5 minutes)

1. **First Response (Positive Experience)**
   - Click "View" on the form
   - Fill in the form with positive feedback
   - Example:
     - Name: Alex Johnson
     - Email: alex@example.com
     - Overall Rating: 5/5
     - Best Session: "The AI in Practice workshop was fantastic!"
     - Suggestions: "More hands-on workshops next time"
     - Likely to Attend Again: Definitely
   - Submit the form

2. **Second Response (Neutral Experience)**
   - Click "View" again to submit another response
   - Fill in with mixed feedback
   - Example:
     - Name: Sam Wilson
     - Email: sam@example.com
     - Overall Rating: 3/5
     - Best Session: "Keynote was interesting but too long"
     - Suggestions: "Better time management for sessions"
     - Likely to Attend Again: Maybe
   - Submit the form

3. **Third Response (Negative Experience)**
   - Click "View" one more time
   - Fill in with constructive criticism
   - Example:
     - Name: Jordan Lee
     - Email: jordan@example.com
     - Overall Rating: 2/5
     - Best Session: "Networking session was good"
     - Suggestions: "More technical depth in presentations, better food options"
     - Likely to Attend Again: Unlikely
   - Submit the form

### 4. Analyzing the Results (4-5 minutes)

1. **Access Analytics**
   - Go back to the dashboard
   - Click "Analytics" for the conference feedback form

2. **Review Summary Metrics**
   - Show the total number of responses (3)
   - Point out the completion rate (100% if all fields were completed)
   - Note the average rating (should be around 3.3/5)

3. **Explore Visualizations**
   - Show the distribution of overall ratings in the pie chart
   - Highlight how the bar chart shows response patterns
   - Point out the text responses and how they're categorized

4. **Filtering and Exporting**
   - Demonstrate how to filter responses (if applicable)
   - Click "Export to CSV" to show how data can be downloaded for further analysis

### 5. Real-time Updates (1-2 minutes)

1. **Submit Another Response**
   - Open an incognito window
   - Submit one more response with different data
   - Show how the analytics update in real-time

### 6. Advanced Features (2-3 minutes, if time permits)

1. **Form Customization**
   - Go back to form builder
   - Add a new question: "What topics would you like to see next year?" (textarea)
   - Make a field required/not required
   - Show how changes are reflected

2. **AI-Powered Insights**
   - If implemented, show the AI analysis of open-ended responses
   - Highlight any sentiment analysis or theme detection

### 7. Q&A (Remaining Time)

> "This concludes our demo. The form is now live and collecting responses. The analytics dashboard will continue to update as more people submit their feedback. Does anyone have any questions about the form builder, response collection, or analytics features?"

## Demo Tips

- **Be Prepared**: Have the form pre-created if short on time
- **Use Real Data**: Make responses realistic and varied
- **Tell a Story**: Frame the demo around improving the conference experience
- **Highlight Key Features**: Emphasize the AI generation and analytics
- **Have a Backup**: Be ready to manually create a form if the AI generation fails

## Troubleshooting

- If the AI is slow, have a pre-generated form ready
- If responses don't appear, check the backend logs
- If charts don't load, ensure all dependencies are installed

## Conclusion

This demo showcases how the AI Form Builder can be used to quickly create professional forms, collect responses, and gain insights through powerful analytics. The platform is designed to be intuitive while offering advanced features for more complex use cases.
